## 積分を解くときに考えること

+ お決まりの形 → 置換してみる
+ 対称性 → King Property のような積分範囲が一致する置換 → 足し引き・同型出現
    - 積分範囲から置換を予測してみる ( ex. \\(\disp \int_{1/2}^2 f(x) ~ dx\\) を \\(t = 1/x\\) と置換する )
+ 塊っぽいものを置換してみる
    - 分母・根号は外したい。
    - 置換で消える分を加味した置換も考える。
    - \\(\disp t = \sqrt[n]{\frac{ax+b}{cx+d}}\\) といった逆関数が書ける置換。
+ 部分積分してみる
    - 同型出現も意識しておく → 積分漸化式になることも
